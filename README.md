#EXCEL SALES PROJECT
#  Vrinda Store Sales Analysis

## Project Overview
This project analyzes the Vrinda Store sales dataset using Microsoft Excel. The dashboard provides insights into sales performance, customer behavior, order trends, and regional performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Dashboard Features
- Sales by Month
- Sales by State
- Sales by Category
- Sales by Gender
- Sales by Age Group
- Order Status Analysis
- Channel-wise Sales Performance

## Key Insights
- Identified top-performing states and sales channels.
- Analyzed customer demographics based on age and gender.
- Compared monthly sales trends.
- Evaluated category-wise sales performance.
